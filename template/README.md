# Pentafox TodoApp

A modern React application built with Vite, TypeScript, Mantine UI, React Router, Zustand, and React Query.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
